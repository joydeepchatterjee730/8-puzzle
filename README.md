# 8-Puzzle Game

A simple 8-puzzle game implemented in Python.

## How to Play
1. Run `main.py`.
2. Use the following keys to move the tiles:
   - `u`: Move up
   - `d`: Move down
   - `l`: Move left
   - `r`: Move right
3. Rearrange the tiles to match the goal state.

## Requirements
- Python 3.x

## Installation
1. Clone the repository.
2. Run the game:
   ```bash
   python main.py
